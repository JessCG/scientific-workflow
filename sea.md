Local network directory structure used by the MOD group at Scripps. 

```
CruiseName/
├── analysis/             <-- Analysis of general interest
├── scripts/              <-- Useful pieces of code
├── papers/               <-- Relevant scientific papers
├── figures/              <-- Figures of general interest
├── cruise_log/           <-- Logs and notes
└── data/                 <-- All raw and processed data for every instrument
    ├── ctd/              <-- A specific instrument
    │   ├── code/         <-- Processing code
    │   ├── raw/          <-- Raw data
    │   └── processed/    <-- Processed data
    └── adcp/             <-- Another instrument
        ├── code/
        ├── raw/
        └── processed/
```
